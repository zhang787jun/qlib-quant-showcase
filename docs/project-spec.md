# Public Quant Showcase Specification

This repository is being shaped into two public GitHub-ready quant projects.

## Project 1: Cross-Sectional Factor Research

Question: do simple cross-sectional factors have useful forward-return signal in a diversified ETF-style universe?

Deliverables:

- Factor table with momentum, reversal, low-volatility, and trend-strength factors.
- Information coefficient summary.
- Quantile forward-return summary.
- Markdown report generated by `scripts/run_factor_research.py`.

Default command:

```bash
PYTHONPATH=src python3 scripts/run_factor_research.py --source sample
```

## Project 2: Dual-Momentum Strategy Backtest

Question: can a simple absolute and cross-sectional momentum rule improve risk-adjusted returns versus an equal-weight benchmark?

Deliverables:

- Monthly rebalanced strategy weights.
- Daily equity curve.
- Performance summary with CAGR, volatility, Sharpe, max drawdown, Calmar, and hit rate.
- Markdown report generated by `scripts/run_strategy_backtest.py`.

Default command:

```bash
PYTHONPATH=src python3 scripts/run_strategy_backtest.py --source sample
```

## Data policy

The default dataset is deterministic synthetic sample data. This keeps CI and public GitHub usage reproducible without leaking private holdings or requiring paid market data. For local exploration, scripts can try Stooq public prices with `--source stooq`; if the network call fails, they fall back to sample data.

## Non-goals for v0.1

- No live trading.
- No private account or holdings data.
- No performance claims suitable for capital allocation.
- No Qlib heavy data dependency yet; Qlib integration can be added after the public minimum projects are stable.
